# 功能
- [X] 登录
- [X] 可指定多个医生
- [X] 可指定就诊人
- [X] 可指定多个挂号时间
- [X] 可指定抢号频率
- [X] 定时抢号
- [X] 定时抢号时间范围
- [ ] MAC桌面端APP
- [ ] Windows桌面端APP
- [ ] 脚本刷号

# 使用

安装依赖
```bash
pip3 install -r requirements.txt
```
